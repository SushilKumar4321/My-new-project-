# Binary Classification Project

This project demonstrates a simple **Binary Classification** approach using Python.  
Binary classification is a type of machine learning task where the goal is to classify data into **two possible classes** (e.g., Yes/No, 0/1, Spam/Not Spam).

## üß† What is Binary Classification?
Binary classification is a supervised learning problem where:
- The output variable has **only two possible values**
- Common examples include:
  - Email: Spam / Not Spam
  - Disease: Positive / Negative
  - Customer: Churn / No Churn
